# Q2 2026

## Impact & Deliverables

- **Fixed a 15x blind spot in Perplexity API cost tracking (2026-06-18):** Asked Claude about research-queue ROI, then caught its first-pass spend estimate ($0.05-0.07) against the real Perplexity billing dashboard ($1.05/30 days) — a 15x gap. Root-caused to a wrong default model (`sonar-pro` at $15/M output tokens instead of plain `sonar` at $1/M) plus two of three research scripts never logging to the spend ledger at all. Fix shipped across `obsidian-second-brain`: real per-token cost accounting, corrected default model, and ledger logging wired into the scripts that were silently skipping it. Generalized into a 4-rule cost-optimization framework applicable to any paid API (Perplexity, Grok, Gemini, future ones). See [[Key Decisions#Research & Intelligence Infrastructure]], [[Gotchas#Perplexity / API cost tracking]], [[api-decision-framework#Cost Optimization Strategy (applies to ANY paid API — Perplexity, Grok, Gemini, future ones)]].

## Technical Growth

- **"Measure, don't estimate":** didn't accept Claude's first-pass cost estimate just because it had a plausible-sounding cost table behind it — checked it against ground truth (the actual billing dashboard) and pushed back ("you sure????") until the real root cause surfaced. The fix that resulted was a real correctness bug (most calls weren't logged at all), not just a config tweak.
